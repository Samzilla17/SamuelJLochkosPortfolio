Some Projects I've Worked On

This will be some information on what I have worked on and you can find these in the file I have given you with all the projects.

Autokey Password Manager

This is an assignment that I made with a group earlier this year focusing on digital literacy and think about ways to improve it in Kenay. This is a device that allows the users to access their MPESA accounts without having to input a password every time. Instead they would walk up and insert their device into the machine and it would grant them access to their accountwith a 2FA code being asked via text message.

This was a collaborative work and we all worked very hard throughout the semester to get this done and refined to where we have it now.

D2LAR

This was an idea I had in my prototyping class which was D2L which is the program we use for school to turn everything in and get updates from our professors. As students we all agree that D2L is not very good and there are a lot of useless things that no one uses. So me and a couple groupmates made an augmented reality version that we think could also work as the new website with much easier and better UX design for students. We also did research to see what students prefered and it was mostly the one we made due to the much easier legibilty and options like a customizable calander.

This was another semester long project that I worked on in my senior year which Professor Wyche gave high praise for the quality of it.
